# 🎵 Music Generator using Python

## 📌 Project Description
This project is a simple AI-based Music Generator developed using Python and the music21 library.

The program generates a melody using predefined musical notes and saves it as a MIDI music file.

---

## 🚀 Features
- Generates simple melodies
- Creates MIDI output file
- Beginner-friendly project
- Uses Python music21 library

---

## 🛠️ Technologies Used
- Python
- music21

---

## ▶️ How to Run

### Install dependencies
```bash
pip install music21
```

### Run the program
```bash
python music_gen.py
```

---

## 🎵 Output
The program generates:
```bash
output.mid
```
